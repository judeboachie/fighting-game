https://judeboachie.github.io/fighting-game/

A 2 player fighting game made with HTML Canvas.

PLAYER 1 CONTROLS: <br/>
A - Move left <br/>
D - Move right <br/>
W - Jump <br/>
SPACE - Attack <br/>



PLAYER 2 CONTROLS: <br/>
LEFT KEY - Move left <br/>
RIGHT KEY - Move right <br/>
UP KEY - Jump <br/>
DOWN KEY - Attack <br/>
